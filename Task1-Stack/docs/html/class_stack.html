template<class T >
void Stack< T >::pop ( )

Удаление узла с вершины стека.

Исключения
std::length_errorВ случае, если прошла проверка на пустоту макросом ASSERT_EMPTY().

1 Создание временного указателя на текущий верхний узел (на удаляемый узел).

2 Смещение вершины стека на его следующий узел.

3 Зануление указателя на следующий узел в удаляемом узле.

4 Удаление предыдущего вершинного узла при помощи созданного указателя.

5 Уменьшение количества элементов на единицу.
